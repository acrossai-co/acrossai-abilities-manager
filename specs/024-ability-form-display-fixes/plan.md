## Phase 1 â€” PHP Changes

### P1.1 â€” CHANGE-1: Fix Source Attribution Default (`AcrossAI_Ability_Merger.php:183`)

**Scope**: Single token change â€” remove `(string)` cast, change fallback `'plugin'` â†’ `null`

**Before**:
```php

```

**After**:
```php

```

**Why this is safe**: `AcrossAI_Ability_Registry_Query.php:79` already guards `empty($merged['source'])` and calls `AcrossAI_Ability_Source_Detector::detect()` which correctly returns `'core'` for `core/*` abilities. Abilities with explicit `source = 'plugin'` in meta are unaffected â€” their value is returned as-is.

**PHPStan note**: `get_meta_item()` return type must accept `null` as default â€” confirm method signature; no cast needed.
